## Question 3
What is a  Schema?
Explain using a simple example.
R/
a mental framework or "cognitive structure" that helps people organize,interpret, 
and quickly process information based on past experiences. It acts as a mental shortcut,
such as a "restaurant schema" that includes menus, tables, and servers, 
allowing you to know what to expect without analyzing every detail.
